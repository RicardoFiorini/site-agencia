# Site Agência
Este projeto consiste em uma **landing page institucional** elegante e funcional, ideal para agências que buscam uma presença online profissional. O desenvolvimento foi focado em código limpo e alta performance, utilizando tecnologias fundamentais da web.
## Principais Seções

- **Home:** Introdução com call-to-action impactante.
- **Serviços:** Exposição detalhada das soluções oferecidas.
- **Sobre:** História e valores da agência.
- **Contato:** Canal direto para novos clientes.

## Status de Desenvolvimento
- [x] Estrutura semântica HTML5
- [x] Estilização personalizada com CSS3
- [x] Responsividade total (Mobile-First)
- [x] Otimização de fontes e ícones
- [ ] Implementação de script para envio de e-mail
## Tecnologias Utilizadas
| Tecnologia | Uso |
| --- | --- |
| HTML5 | Marcação e estrutura de conteúdo |
| CSS3 | Layout (Flexbox), cores e tipografia |
| Google Fonts | Integração de fontes modernas |
> [!TIP]
> Este projeto não utiliza frameworks como Bootstrap, o que resulta em um carregamento extremamente rápido e um arquivo CSS totalmente customizado.
